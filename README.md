# 🚀 NOC Ultimate Automation Tool

Ferramenta desenvolvida em Python para facilitar o dia a dia em centros de operações de rede (NOC). Centraliza diagnósticos técnicos que antes eram feitos de forma manual e dispersa.

### 🛠 Funcionalidades
- **Traceroute Inteligente:** Rastreio de rota em tempo real com diagnóstico de ponto de falha.
- **Identificador de Operadora (WHOIS):** Consulta instantânea de ASN, ISP e localização de qualquer IP público.
- **Scanner de Portas:** Verificação rápida de portas críticas (Winbox 8299, Web 9999, SSH 22, 2288, etc).
- **Calculadora PTP:** Cálculo de sub-rede (Network, First/Last Host, Broadcast) com lógica de identificação Par/Ímpar (Operadora/Cliente).
- **Leitor de MAC Vendor:** Identifica o fabricante do equipamento através do endereço MAC.

### 🚀 Como Usar
1. Baixe o executável na seção [Releases](link-da-sua-release).
2. Insira o IP, Bloco ou MAC no campo principal.
3. Escolha a tarefa desejada através dos botões de ação.

### 💻 Tecnologias
- Python 3.10+
- CustomTkinter (Interface Moderna)
- Bibliotecas: Requests, Socket, Ipaddress, Subprocess.
